# Vietnam Rideshare Compare

A web app that compares rideshare prices between **Grab** and **Xanh SM** in Vietnam.

## Features

- Enter your trip distance (in km) and select a vehicle type
- Instantly compare estimated fares side-by-side for Grab and Xanh SM
- "Best Price" badge and savings summary highlight the cheaper option
- Covers **Motorcycle**, **Car 4-seat**, and **Car 7-seat** categories

## Tech Stack

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/) – build tool
- [Tailwind CSS v4](https://tailwindcss.com/) – utility-first styling
- [Radix UI](https://www.radix-ui.com/) – accessible UI primitives (Tabs, Label, Separator, Tooltip)

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

> **Note:** Prices shown are estimates based on approximate 2024 base rates. Always check the official Grab or Xanh SM app for the exact fare before booking.
